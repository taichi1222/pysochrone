# Pysochrone
osmnxを用いて任意の点からIsochrone（等時線）を作成するプログラム

e.g.) 京成稲毛駅からの徒歩10分等時線
```py
from pysochrone import isochrone

isochrone(140.08571, 35.63725, "walk", 10, 80)

### POLYGON ((140.08882 35.63135, 140.08456 35.631...
```
